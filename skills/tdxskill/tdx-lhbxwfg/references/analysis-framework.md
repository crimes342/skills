# 龙虎榜席位风格分析框架

## 表结构速记

- `summary`
  - 用来确认信息类型编码、总买入、总卖出、净额和总成交额
- `details`
  - 用来识别买一到买五、卖一到卖五、同席位双向成交和排名结构
- `seat_profiles`
  - 用来读取营业部标签代码、标签名称
- `extra`
  - 用来补日期、证券代码、交易类型、排名、买卖标签

## 常用判断信号

### 1. 买方主导型

- 特征
  - 买方前两席净买额明显高于其余席位
  - 买入金额集中，卖方分散
- 可写成
  - 当日资金更像由少数核心席位主动点火或主导承接
- 风险提示
  - 单日集中买入不等于次日持续锁仓

### 2. 卖方兑现型

- 特征
  - 卖一到卖三卖出额显著高于买方核心席位
  - 净额转弱或净卖出
- 可写成
  - 席位结构更偏兑现，抛压侧强于承接侧
- 风险提示
  - 若股价仍强，可能只是高位换手，不要直接写成见顶

### 3. 高换手博弈型

- 特征
  - 同一席位或同类席位同时出现在买方和卖方
  - 买卖额都大，净额不突出
- 可写成
  - 资金更像做 T、日内腾挪或高换手博弈，持股意愿未必强
- 风险提示
  - 这种结构通常对次日承接要求更高

### 4. 情绪接力型

- 特征
  - 多个短线活跃席位集中出现在买方
  - 股价处于加速段或题材热度高
- 可写成
  - 龙虎榜呈现明显短线接力特征，交易逻辑偏情绪驱动
- 风险提示
  - 情绪型席位持续性依赖板块强度和次日溢价

### 5. 机构偏配置型

- 特征
  - `seat_profiles` 或席位名称中有明确机构标签
  - 单边买入更明显，双向倒手较少
- 可写成
  - 若标签识别无误，更像机构性质资金在做方向性配置
- 风险提示
  - 只有在标签明确时才这样写，不要把普通营业部硬解释成机构

## 建议配套验证

- 用 `tdx_quotes` 看当日涨跌幅、换手率和盘口强弱
- 用 `tdx_kline` 看日线位置、量价和前高前低关系
- 用 `wenda_news_query` 或 `wenda_notice_query` 看催化是否真实存在
- 用 `tdx_screener` 看同题材是否形成梯队或扩散

## 写作纪律

- 先写数据事实，再写风格判断
- 推断用“更像”“倾向于”“可能”“显示出”
- 不用“确定是”“必然会”“主力已经完全控盘”这类绝对化表述
- 不把龙虎榜席位行为直接等同于未来收益承诺
