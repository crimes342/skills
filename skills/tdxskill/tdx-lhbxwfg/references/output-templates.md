# 输出模板

## 问答模板

```markdown
结论先行：这次龙虎榜更像[主动做多/情绪接力/高位兑现/高换手博弈]。

依据有三点：
1. [净买额/净卖额/总成交额] 显示当日上榜资金整体偏[进攻/兑现]。
2. 买卖席位中，[核心席位名称] 的行为最关键，体现为[集中买入/集中卖出/双向对倒]。
3. 结合股价位置与题材环境，这次上榜更偏[趋势强化/高位分歧/消息刺激后的博弈]。

风险提示：
- 龙虎榜只覆盖上榜席位，不代表全部资金流向。
- 单日席位行为不直接等于后续走势。
```

## 文章模板

```markdown
# [标题]

先写市场背景和当日异动，再引出龙虎榜。

第二段写核心数据：
- 净买额或净卖额
- 关键席位
- 席位是否集中

第三段写席位风格：
- 是谁在主导
- 更像接力还是兑现
- 是否存在高换手或情绪化博弈

第四段写交易含义：
- 对短线情绪的影响
- 对次日博弈的启示
- 需要警惕的风险

结尾回到一句话判断：
- 这是一次[资金进攻/高位分歧/兑现撤退]性质的龙虎榜。
```
