# Decision Rules

## 情绪周期判定

按证据强弱做阶段判断，不要只凭单一现象下结论。

| 阶段 | 典型特征 | 交易动作 |
|---|---|---|
| 冰点 | 跌停压力大，高位股负反馈明显，连板高度压缩，涨停虽有但承接弱 | 以观察为主，只做最强辨识度标的试错 |
| 修复 | 跌停减少，高位股止跌，龙头出现承接，低位首板开始扩散 | 关注弱转强、分歧修复，不做后排乱追 |
| 主升 | 连板高度抬升，核心龙头和中军共振，题材有梯队，回流明显 | 可以围绕龙头、中军和核心分支积极博弈 |
| 高位震荡 | 高位股频繁分歧，板块内部轮动加快，前排还能撑住但后排掉队 | 只做前排核心，不做一致性后排接力 |
| 退潮 | 高位股补跌，梯队断层，轮动杂乱，涨停多为脉冲 | 降低预期，避免打板接力，优先防守 |

## 角色划分

| 角色 | 判定重点 | 常见博弈方式 |
|---|---|---|
| 情绪龙头 | 辨识度最高，最能带动同题材情绪，分歧时仍有承接 | 弱转强、分歧转一致、核心回封 |
| 趋势中军 | 容量大、成交强、趋势稳，常承担资金中枢作用 | 趋势低吸、分歧承接、放量新高 |
| 补涨 | 位置偏低，通常在龙头打开空间后才被资金挖掘 | 卡位、补涨轮动、低位启动 |
| 后排跟风 | 多数依赖板块扩散，独立性弱，容易掉队 | 只做最强时点，不宜重仓恋战 |
| 伪强 | 表面强势，但无题材梯队、无资金持续、无辨识度支撑 | 以回避为主 |

## 龙头博弈标签

用以下标签描述博弈状态，避免空泛评价：

- `弱转强`：前一交易日分歧，下一交易日主动转强并获得承接。
- `分歧转一致`：盘中或隔日从分歧走向统一预期，前排获得强化。
- `卡位`：非原始龙头尝试抢夺市场锚点，必须说明是否真正成功。
- `高位兑现`：高位放量、封单衰减、承接不足，资金兑现特征明显。
- `跟风补涨`：板块强，但地位弱，更多承担扩散作用。

## 明日观察点模板

优先输出具体观察点，不要只写“看市场反馈”：

1. 看龙头是否继续给溢价，尤其是弱转强是否成立。
2. 看中军是否放量新高，确认板块不是纯情绪脉冲。
3. 看后排是否大面积掉队，验证梯队是否健康。
4. 看高位股负反馈是否扩散到同题材其他个股。
5. 看新首板是否继续围绕主线扩散，而不是题材乱轮动。
