# 问小达选基金 使用指南

## 技能概述

本技能提供公募基金智能筛选能力，帮助用户快速找到符合特定条件的基金。通过自然语言查询，支持多种筛选维度的组合。

## 核心功能

### 支持的筛选维度

| 维度 | 说明 | 示例 |
|------|------|------|
| **基金类型** | 按股票型、债券型、混合型、货币型等筛选 | 股票型基金、债券型基金、QDII |
| **业绩筛选** | 按收益率、回撤、夏普比率等筛选 | 近一年收益率前10、最大回撤小于5% |
| **基金经理** | 按基金经理查询 | 某某经理管理的基金 |
| **风险指标** | 按波动率、最大回撤等筛选 | 低波动率基金、风险控制能力强的基金 |
| **持仓分析** | 按重仓股、行业配置等筛选 | 重仓白酒的基金、科技配置比例高的基金 |
| **资产配置** | 按股票仓位、债券仓位等筛选 | 股票仓位高于80%、均衡配置 |
| **多条件组合** | 多个条件组合筛选 | 股票型且近一年收益前10 |

### 数据来源

- **主要数据源**: 通达信问小达 (tdx_screener)
- **辅助数据源**: 通达信行情数据、资讯数据

## 使用流程

### 步骤 1: 接收用户查询

接收用户的自然语言基金筛选请求，分析用户意图和筛选条件。

**用户输入示例**:
- "近一年收益率前10的股票型基金"
- "某经理管理的基金"
- "低风险且收益稳定的债券基金"

### 步骤 2: Query 改写与标准化

将用户的口语化表达改写为标准的金融查询语句。

**改写规则**:
- 保留核心意图（基金类型、收益率、基金经理等）
- 将口语转为标准术语（"给我选一个基金" → "基金有哪些"）
- 适当简化复杂条件
- 保持原意不变

**标准化示例**:
| 用户输入 | 标准化 Query |
|---------|---------------|
| 帮我选个收益好的股票型基金 | 近一年收益率前10的股票型基金 |
| 某经理管的基金 | 某某经理管理的基金 |
| 不想亏太多的债券基金 | 低风险的债券基金 |

### 步骤 3: 调用数据接口

使用 `tdx_screener` 工具查询基金数据：

```bash
tdx_screener message="<标准化的Query>" rang="JJ"
```

**参数说明**:
- `message`: 标准化后的查询语句
- `rang`: 市场范围，基金查询使用 "JJ"

### 步骤 4: 数据扩展决策（可选）

根据数据完整性决定是否需要补充信息：

**数据完整**: 直接格式化返回结果

**需要扩展背景信息**:
- 调用 `wenda_news_query` 获取基金相关资讯
- 调用 `wenda_report_query` 获取基金研报观点
- 调用 `tdx_quotes` 获取实时行情数据

### 步骤 5: 格式化输出

组织语言回答用户，确保结果清晰易懂。

**输出格式要求**:
- 使用表格展示基金列表（代码、名称、收益率、规模等）
- 说明数据来源
- 如果改写了问句，需说明最终使用的查询条件

## 注意事项

### 重要提示

1. **数据来源标注**: 所有引用通达信数据的地方，必须强调**数据来源于通达信问小达**

2. **数据有效性**: 基金数据可能存在 T+1 延迟，净值数据通常按交易日更新

3. **查询限制**:
   - 单次查询最多返回一定数量的结果
   - 复杂条件可能需要分步骤查询
   - 部分基金可能暂停申购或赎回

4. **风险提示**:
   - 历史业绩不代表未来表现
   - 基金投资有风险，需根据自身风险承受能力选择

5. **错误处理**: 
   - 如果 API 调用失败，给出友好错误提示
   - 如果无数据返回，提示条件可能过于严格，建议放宽条件或访问通达信客户端
   - 最多重试 2 次，逐步放宽条件

### 用户引导

当无法获取数据或查询失败时，提示用户：

**数据来源**: 通达信
**客户端下载**: https://www.tdx.com.cn/

## 错误处理

### 常见错误场景

| 错误类型 | 处理方式 |
|----------|----------|
| API 调用失败 | 显示错误信息，提示稍后重试 |
| 无数据返回 | 提示条件可能过于严格，建议放宽条件或访问通达信客户端 |
| 基金代码错误 | 提示检查基金代码是否正确 |
| 网络超时 | 提示检查网络连接，建议重试 |
| 条件过于复杂 | 拆分为多个简单条件分别查询 |

### 重试策略

- **重试次数**: 最多 2 次
- **放宽条件**: 每次重试适当放宽筛选条件
- **降级方案**: 引导用户使用通达信客户端查询

## 技能调用约束

### 必须使用本技能的场景

当用户询问以下类型问题时，**必须**使用此技能：

- 基金筛选相关（"收益好的基金"、"低风险基金"）
- 基金类型查询（"股票型基金"、"债券型基金"）
- 基金经理查询（"某经理管理的基金"）
- 基金业绩排行（"近一年收益前10"）
- 基金组合条件筛选（"收益好且低风险"）
- 特定类型基金（"QDII基金"、"ETF基金"）

### 不适用场景

- 个股筛选（应使用【问小达选A股】）
- 板块筛选（应使用【问小达选板块】）
- ETF 筛选（应使用【问小达选ETF】）
- 基金经理筛选（应使用【问小达选基金经理】）
- 基金公司筛选（应使用【问小达选基金公司】）

## 基金类型说明

### 常见基金类型

| 类型 | 说明 | 风险等级 |
|------|------|----------|
| 股票型 | 主要投资于股票的基金 | 高 |
| 债券型 | 主要投资于债券的基金 | 低 |
| 混合型 | 同时投资股票和债券的基金 | 中 |
| 货币型 | 投资于货币市场工具的基金 | 极低 |
| QDII | 投资于海外市场的基金 | 中-高 |
| 指数型 | 跟踪特定指数的基金 | 中 |
| ETF | 交易所交易的指数基金 | 中 |

### 业绩指标说明

| 指标 | 说明 |
|------|------|
| 近一年收益 | 最近一个会计年度的收益率 |
| 近三年收益 | 最近三个会计年度的年化收益率 |
| 最大回撤 | 基金净值从高点回落的最大幅度 |
| 夏普比率 | 单位风险的超额收益，越高越好 |
| 波动率 | 净值波动的程度 |
| 阿尔法 | 选股能力的量化指标 |
